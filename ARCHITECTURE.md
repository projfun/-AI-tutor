## 🔐 Процесс авторизации

```
1. Пользователь получает токен из браузера
   (Authorization: Bearer <token>)
   
2. Вставляет в приложение
   request: set_school_portal_token(token)
   
3. Бэкенд проверяет валидность
   GET /api/tags (проверка наличия учебника)
   
4. Если успешно:
   • Сохраняем токен в .env
   • Загружаем профиль и расписание
   • Переключаемся на основной интерфейс
   
5. Токен используется для всех последующих запросов
   Authorization: Bearer <token>
```

## 💭 Жизненный цикл вопроса

```
[1] Пользователь вводит вопрос
    ↓
[2] Фронтенд отправляет ask_tutor(question, subject)
    ↓
[3] Бэкенд начинает обработку
    ├─ Поиск в интернете (DuckDuckGo)
    │  └─ Получение 3 релевантных результатов
    │
    ├─ Поиск в RAG (учебники)
    │  └─ Получение контекста из загруженных PDF
    │
    └─ Добавление контекста к вопросу
    ↓
[4] Формирование промпта
    ├─ System Prompt (рол, правила)
    ├─ История разговора (предыдущие вопросы)
    ├─ Контекст из интернета
    ├─ Контекст из учебников
    └─ Текущий вопрос
    ↓
[5] Отправка в Ollama API
    POST http://localhost:11434/api/chat
    ├─ Model: llama3.1:8b-q4_K_M
    ├─ Stream: true (потоковая передача)
    └─ Messages: [...]
    ↓
[6] Ollama генерирует ответ
    ├─ Обработка System Prompt
    ├─ Анализ контекста  
    ├─ Генерация токенов
    └─ Потоковое отправление JSON
    ↓
[7] Бэкенд принимает и перенаправляет
    ├─ Парсит JSON из Ollama
    ├─ Извлекает текст ответа
    └─ Отправляет в Eel.js update_tutor_response()
    ↓
[8] Фронтенд обновляет интерфейс в реальном времени
    ├─ Добавляет чанки текста в чат
    ├─ Форматирует Markdown в HTML
    └─ Прокручивает к последнему сообщению
    ↓
[9] Сохранение в историю
    ├─ Добавляет вопрос в conversation_history
    └─ Добавляет ответ в conversation_history
    ↓
✅ Готово!
```

## 🤖 Система промпта

### System Prompt (Основные правила)

```python
SYSTEM_PROMPT = """
Ты — ИИ-тьютор для ученика
Правила:
1. Никогда не решай задачи за ученика
2. Объясняй алгоритм, не ответ
3. Используй примеры и аналогии
4. Проверяй понимание [вопрос к ученику]
5. Дай ссылки на источники
"""
```

### Полный контекст к вопросу

```
SYSTEM_PROMPT
    ↓
[История разговора]
    Вопрос 1: "Как интегрировать?"
    Ответ 1: "Формула интегрирования..."
    Вопрос 2: "А это работает для...?"
    ↓
[Контекст из интернета]
    Из Wikipedia: "...определение интеграла..."
    Из Khan Academy: "...пошаговые примеры..."
    ↓
[Контекст из учебников]
    Из Учебника Математика 9: "...теорема о интегралах..."
    ↓
[Текущий вопрос]
    "Как решить интеграл такого вида?"
```

## 🔌 Eel.js: Связь между Python и JavaScript

### Вызовы из JavaScript в Python

```javascript
// Фронтенд вызывает бэкенд
eel.ask_tutor(question, subject)();

// Это становится в Python:
@eel.expose
def ask_tutor(question, subject=None):
    # ...обработка...
    return {"response": answer}
```

### Обновления из Python в JavaScript

```python
# Бэкенд отправляет обновление
eel.update_tutor_response(chunk)

# Фронтенд этот обработчик
<script>
eel._update_tutor_response = function(chunk) {
    // ...добавляем чанк в чат...
}
</script>
```

## 🚨 Обработка ошибок

```python
try:
    result = eel.ask_tutor(question)()
except requests.exceptions.Timeout:
    # Ollama не отвечает
    return "⏱️ Ошибка: превышено время"
except requests.exceptions.ConnectionError:
    # Нет соединения с Ollama
    return "❌ Ошибка: Ollama не запущена"
except Exception as e:
    # Неизвестная ошибка
    return f"❌ Ошибка: {str(e)}"
```
