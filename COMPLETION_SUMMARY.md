## 🎯 ИНСТРУКЦИЯ ДЛЯ СТАРТЕРА

### Шаг 1: Запуск приложения (30 сек)

```bash
# Способ 1 (Windows) — двойной клик на:
run.bat

# Способ 2 (кроссплатформа)
python run.py

# Способ 3
python main.py
```

### Шаг 2: Первая инициализация (10-20 минут)

**Что происходит автоматически:**
1. ✅ Проверка Ollama
2. ✅ Скачивание и установка Ollama (если нет)
3. ✅ Запуск Ollama сервера
4. ✅ Загрузка модели llama3.1:8b-q4_K_M (~4.5 ГБ)
5. ✅ Открытие браузера на http://localhost:8000

### Шаг 3: Авторизация (5 минут)

1. Откройте школьный портал в новой вкладке
2. Скопируйте токен (F12 → Network → любой запрос → Authorization)
3. Вставьте в приложение → нажмите "Подтвердить"
4. ✅ **Готово!** Расписание загружено

### Шаг 4: Первый вопрос (1 минута)

1. Выберите предмет (например, "Физика")
2. Напишите вопрос (например, "Объясни закон Ома")
3. Нажмите "📤 Отправить" или Ctrl+Enter
4. ⏳ Ответ придет за 3-30 секунд

## 🆘 ЧТО ДЕЛАТЬ ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ

### Проблема: "Ollama not found"
```bash
# Решение 1: Установите вручную с https://ollama.ai
# Решение 2: Запустите скрипт
python env_setup.py
```

### Проблема: "Token invalid"
```
1. Откройте портал заново (новая вкладка браузера)
2. Повторите копирование токена
3. Вставьте его в приложение
```

### Проблема: "Очень медленно"
```
1. Закройте ненужные приложения
2. В config.py используйте легкую модель:
   DEFAULT_MODEL = MODELS["light"]  # Qwen 4B
```

### Проблема: "Порт 8000 занят"
```
1. Найдите процесс: netstat -ano | findstr 8000
2. Или измените порт в config.py:
   WEB_CONFIG = {"port": 8080}
```
