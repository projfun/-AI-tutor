## 🎯 Ключевые файлы

### 🔥 main.py - Ядро приложения
```python
# Основные функции:
- Инициализация Eel интерфейса
- Настройка AI движка
- Управление порталом
- Обработка запросов от фронтенда
```

### 🤖 engine.py - AI движок
```python
# Основные компоненты:
- Ollama интеграция
- Система промптов
- RAG интеграция
- Веб-поиск
```

### 🏫 portal.py - Школьный портал
```python
# Основные функции:
- Авторизация по токену
- Получение расписания
- Загрузка домашних заданий
- BrowserConnector для сложных случаев
```

### 🌐 web/ - Веб-интерфейс
```
index.html          # Основная страница с чатом и расписанием
style.css           # Стили TailwindCSS + кастомные
script.js           # Логика интерфейса, обработка Eel вызовов
```

## 🔄 Потоки данных

### 📥 Входящие данные
```
1. 🎫 Токен авторизации → Portal Client
2. 📅 Расписание уроков → UI + RAG
3. 📚 PDF учебники → Vector Database
4. ❓ Вопрос ученика → AI Engine
```

### 📤 Исходящие данные
```
1. 🤖 Ответ ИИ → UI (потоково)
2. 📊 Прогресс обучения → Analytics
3. 💾 Кэшированные ответы → Local Storage
4. 🔗 Источники → UI (ссылки)
```

## ⚙️ Конфигурация


## 🚀 Развертывание

### 🏠 Локальный запуск
```bash
# 1. Установка зависимостей
pip install -r requirements.txt

# 2. Запуск приложения
python main.py
# или
run.bat  # Windows
```

### 📦 Сборка в исполняемый файл
```bash
# PyInstaller сборка
python build.py

# Результат:
dist/ai-tutor.exe    # Windows
dist/ai-tutor        # Linux/macOS
```
