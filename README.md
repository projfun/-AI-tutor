## 🎯 Суть проекта

AI-Tutor — это десктопное приложение, которое превращает локальную языковую модель в персонального наставника. Приложение интегрируется с порталом «Моя Школа», знает расписание ученика и помогает разобраться в сложных темах через **объяснение алгоритмов**, а не дачу готовых ответов.

### ✨ Ключевые особенности

- **🔒 Локальность**: Весь ИИ работает на ПК пользователя (Ollama)
- **👨‍🏫 Педагогический подход**: ИИ объясняет теорию и дает алгоритм, не решая задачу
- **📅 Интеграция с портальном**: Автоматическая загрузка расписания из "Моя Школа"
- **⚡ Скорость**: Использование легких моделей (4B-8B параметров)
- **🌐 Веб-поиск**: Интеграция с интернетом для актуальной информации
- **📚 RAG база знаний**: Поиск в загруженных учебниках (PDF)

## 🚀 Быстрый старт

### Минимальные требования

- **ОС**: Windows 10+, macOS, Linux
- **ОЗУ**: 8 ГБ (минимум 4 ГБ для легких моделей)
- **Диск**: 10 ГБ (для модели + Ubuntu)
- **Python**: 3.13+

### Установка

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/your-repo/ai-tutor.git
   cd ai-tutor
   ```

2. **Установите зависимости**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустите приложение**
   ```bash
   python main.py
   ```

   Приложение **автоматически**:
   - Проверит наличие Ollama
   - Скачает и установит её (если нужно)
   - Загрузит модель Llama 3.1 8B
   - Откроет веб-интерфейс в браузере

### Авторизация в школьном портале

1. Откройте [портал "Моя Школа"](https://authedu.mosreg.ru)
2. Нажмите **F12** → вкладка **Network**
3. Обновите страницу (F5)
4. Найдите любой запрос и скопируйте заголовок **Authorization**
5. Вставьте токен в приложение

## 🤖 Как работает AI-Наставник

### System Prompt (Жесткие правила)

```
Ты — ИИ-тьютор для ученика
Правила:
1. Никогда не решай задачи за ученика
2. Объясняй алгоритм, не ответ
3. Используй примеры и аналогии
4. Проверяй понимание
5. Дай ссылки на источники
```

### Пример диалога

**Ученик:** "Как решить квадратное уравнение?"

**Наставник:** 
```
📚 Теория: Квадратное уравнение — это уравнение вида ax² + bx + c = 0

🎯 Алгоритм решения:
1. Определи коэффициенты a, b, c
2. Вычисли дискриминант: D = b² - 4ac
3. Если D > 0: два корня. D = 0: один корень. D < 0: нет корней
4. Используй формулу: x = (-b ± √D) / 2a

💡 Пример: 2x² + 3x + 1 = 0
   D = 9 - 8 = 1, √D = 1, x₁ = (-3+1)/4 = -0.5, x₂ = (-3-1)/4 = -1

📖 Подробнее: [Khan Academy](link), [Википедия](link)

❓ А теперь ты реши: 3x² - 5x + 2 = 0
```

## 🚨 Известные проблемы

1. **Ollama не запускается на Windows**
   - Решение: Установите вручную с https://ollama.ai

2. **Модель загружается медленно**
   - Решение: Используйте квантованную версию (q4_K_M)

3. **Ошибка при авторизации портала**
   - Решение: Убедитесь, что токен скопирован полностью без пробелов
